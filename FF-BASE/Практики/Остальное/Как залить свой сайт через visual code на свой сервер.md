как залить свой сайт через visual code: 
- авторизуйтесь в firebase через терминал.
 - заменить папку web на эти данные из архива 
 - изменить данные индекса на свои. 
 - логотип можно заменить: открывается логотип в svg формате через блокнот  и копируете данные в индекс (gpt попросить заменить), так же ключ карт прописать не забудьте. 
 - замените все логотипы на свои (сохраняяя названия!!!) 
 - в файле manifest тоже на свои данные замените.  
 - далее pubspec ( заменить так что бы версия пакетов соотвествовала, вам ненужны пакеты, но когда скачиваете код там у одного пакета старая версия и надо сменить там определенные данные) 
 - файл firebase нужно поместить в общую папку (где  pubspec) файл upload_data дополненная функция что бы можно было загружать айфоновские фотки которые обычный пакет не поддерживает. просто замените ваш C:\Users\cabit\Desktop\папкка с кодом\lib\flutter_flow в этом пути.  
 - для публикуации пусты файлы flutter build web --release firebase deploy --only hosting созданы что бы просто команды для терминала копировать.
